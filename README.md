# WBE Graph Modelling

This repository focuses on graph-based modelling for wastewater-based epidemiology (WBE), with an emphasis on spatio-temporal prediction of viral spread.

## Scope
The project explores different modelling approaches, including:
- XGBoost
- LSTM
- STGCN

## Current Work
- Replication and extension of models from Sun’s MSc dissertation
- Understanding data pipelines and feature construction
- Benchmarking different modelling approaches
